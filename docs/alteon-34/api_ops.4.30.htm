Export GSLB DNSSEC Zone
REST Method 
GET
Description 
Exports a DNSSEC zone from Alteon to the client computer.
Alteon Relative URI Path 
/config/getdnszone?id=<value>&src=<txt/file>&type=<ds/dnskey>
Example
Request 
 
https://172.17.151.225/config/getdnszone?id=zone1&src=file&type=dnskey&key=z1
Response 
 
{
"status":"ok",
}
