Enable Packet Capture
REST Method 
POST
Description 
Manages Alteon packet capture behavior.
Alteon Relative URI Path 
Start capturing:
/config/pktcapture?cmd=start&port=<value>&cmdstr=<string>&vlan=<value>&len=<value>&count=<value> where:
— port specifies the device physical port. Supports port range 1-2, or discrete values, where “m” traps the management traffic.
— cmdstr specifies the pcap command filter.
— vlan specifies the VLAN ID.
— len specifies the packet capture length in bytes.
— count specifies the number of records to save.
Stop capturing:
/config/pktcapture?cmd=stop
Clear the capture log:
/config/pktcapture?cmd=clear
Example
Request 
 
https://172.17.151.225/config/pktcapture?cmd=start&port=1-3,m,5,6&cmdstr=
src host 1.1.1.1 dst port 80&vlan=2&len=6&count=20
Response 
 
{
"status":"ok",
"message":"200 OK - The request has succeeded"
