Note: In the examples in this section:
Alteon URL paths are relative to the URL prefix https://<device IP address>/.
The URL prefix is usually followed by the config Web resource name
For example, https://<device IP address>/config/<object name>
Parameters in a URL path are marked by single angle quotation marks. For example, <device_name>.
Note: The Alteon REST API requires that you add a line break (“\n”) to JSON payloads.
For example:
JSON_payload =
{‘x’: 7,
‘y’: 9}
A JSON payload without a line break will not work properly.
For example: JSON_payload =
{‘x’: 7, ‘y’: 9}
