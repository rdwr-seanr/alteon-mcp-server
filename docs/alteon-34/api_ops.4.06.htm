Get Single Table Item
HTTP Method 
GET
Description 
Requests the contents of a table.
Returns JSON text describing the content of a single table row, and also when there are single or dual indexes for the table. For a sample response, see JSON Data Formats.
Alteon Relative URI Path 
/config/<table name> returns the entire table.
/config/<table name>/<table key1> returns a specified row in a table with a single index where:
<table key> specifies the row to read from. If no value is specified, Alteon reads the entire table.
/config/<table name>/<table key1>/<table key2> returns a specified row in a table with a double index.
/config/<table name>/<table key1>/<table key2>?&props=<table element1>,<table element2>,<table element3> where the following optional parameters are specified:
props specifies the columns to read from the specified entry. If no value is specified, Alteon reads all columns.
Example 
Retrieves the entry with VLAN ID 2 from the VLAN table and specifies a list of required columns.
Request 
 
https://172.17.151.225/config/VlanNewCfgTable/2/?&props=State,VlanId,VlanName,Ports
Response 
 
{
"VlanNewCfgTable":[{
"State":"2",
"VlanId":"2",
"VlanName":"Default VLAN 2",
"Ports":"20:00:00:00:00:00:00:00"
}]
}
