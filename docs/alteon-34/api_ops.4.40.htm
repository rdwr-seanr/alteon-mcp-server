Purge HTTP Cache
REST Method 
PUT
Description 
Purges the cached content of the relevant HTTP responses.
Alteon Relative URI Path 
/config/SlbOperEnhHttpCachePurgeTable/<ServerIndex>/<ServiceIndex>/
Example
Request 
 
https://172.17.151.225/config/SlbOperEnhHttpCachePurgeTable/ll/80/
Body 
 
{
"ServerIndex":"ll",
"ServiceIndex":"80",
"URL":"abcd"
}
Response 
 
{
"status":"ok"
}
