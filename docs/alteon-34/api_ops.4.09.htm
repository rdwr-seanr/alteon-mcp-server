Update Table Item
HTTP Method 
PUT
Description 
Sends an HTTP PUT request with a body containing JSON text describing the changed parameters of the updated row. For a sample response, see JSON Data Formats.
Note: Updating an object that has augmented tables is done separately on each table. For more information about augmented tables, see Working with Augmented Tables.
Alteon Relative URI Path 
/config/<table name>/<table key>
Example 
Updates the client NAT settings for real server “real2”.
Request 
 
https://172.17.151.225/config/SlbNewCfgEnhRealServerSecondPartTable/real2
Body 
 
{
"Index": "real2",
"ProxyIpMode":"2",
"ProxyIpAddr":"66.66.66.66",
"ProxyIpMask":"255.255.0.0"
}
Response 
 
{
"status":"ok"
}
Request (from the real server second part table) 
 
https://172.17.151.225/config/SlbNewCfgEnhRealServerTable/real2
Body 
 
{
"Index": "real2",
"Weight":"10"
}
Response 
 
{
"status":"ok"
}
