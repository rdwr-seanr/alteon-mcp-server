Export Packet Capture File
REST Method 
GET
Description 
Downloads the packet capture file from Alteon to the client computer. The file is available as a regular attachment.
Alteon Relative URI Path 
/config/getcapturefile
Example  
 
https://172.17.151.225/config/getcapturefile
