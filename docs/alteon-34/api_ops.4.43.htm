Associating Multiple Objects
Association of multiple each object association can be done in a separate rest request. (It cannot be done in a single REST request)
You cannot use a single REST request to associate multiple objects. You must usue a separate request for each association. For example, use separate PUT requests to associate multiple ports to a VLAN, as follows:
To associate multiple ports to a VLANÂ 
1. Create the VLAN with request URL https://<device ID>/config/ VlanNewCfgTable/<vlanID>/.
2. Associate ports to the VLAN.
Send the following request body for each port:
 
{
"VlanId":"<vlanID>",
"AddPort":"<associated port ID>"
}
3. Create a group with request URL https://<device ID>/config/SlbNewCfgEnhGroupTable/<groupID>/.
4. Send a separate PUT request for each real server to be associated with the group, as follows:
 
{
"Index":"<groupID>",
"AddServer":"<associated real ID>"
}
