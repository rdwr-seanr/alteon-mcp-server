Filter Table Items
HTTP Method 
GET
Description 
Filters table rows according to specified columns and rows starting from the first row.
Alteon Relative URI Path 
/config/<table name>?filter=<table element1>:<value>,<table element2>:<value>&
filtertype=<any or exact>&props=<table element1>,<table element2>,<table element3>,
<table element9>
where:
<table element1:value> specifies the first required element and value. For example, IP address:105.5.5.5.
<table element2:value> specifies the second required element and value. For example, description:myRealServer.
filtertype specifies the type of matching, as follows:
— any — includes a specified string (recommended for strings)
— exact — specifies exact matching for IP addresses and integers; for strings, exact functions as any
props specifies the table columns to be included in the filter response.
Example 
Filters the VLAN table for disabled entries.
Request 
 
https://172.17.151.225/config/VlanNewCfgTable?filter=State:3&filtertype=exact&props=State,VlanId,VlanName,Ports,Ipv6LlaGen,Stg
