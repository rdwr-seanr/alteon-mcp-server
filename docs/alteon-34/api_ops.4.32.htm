Import AppShape++ Script
REST Method 
PUT
Description 
Imports an AppShape++ script to Alteon from the client computer.
Alteon Relative URI Path 
/config/appshapeimport?id=<value>&name=<value>&state=<ena/dis>
Example
Request 
 
https://172.17.151.225/config/appshapeimport?id=111&name=script1&state=ena
Body 
 
when HTTP_REQUEST {
set timestamp [clock milliseconds]
log $timestamp
}
 
when HTTP_RESPONSE {
set reply_time [expr [clock milliseconds] - $timestamp]
log $reply_time
HTTP::header insert Alteon $reply_time
}
-----END
Response 
 
{
"status":"ok",
"message":"200 OK - The request has succeeded"
}
