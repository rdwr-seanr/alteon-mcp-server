Import GSLB DNSSEC Key
REST Method 
PUT
Description 
Imports a DNSSEC key to Alteon from the client computer.
Alteon Relative URI Path 
/config/dnskeyimport?id=<value>&type=<key/zone>&state=<ena/dis>&passphrase=<value>&
keysz=<1024/2048/4096>&hash=<1/256/512>&ttl=<0-86400>&exp=<0-473040000>&rollover=
<0-473040000>&sigval=<3600-473040000>&sigPUB=<3600-473040000> where:
id specifies the key ID.
type specifies whether the key is key-signing or zone-signing.
state specifies whether the key is enabled or disabled.
passphrase specifies the password to decrypt the private key (required when type is set to key or pair).
keysz specifies the size of the key (1024, 2048, or 4096).
hash specifies the hash algorithm.
ttl specifies the key TTL interval in seconds.
exp specifies the key expiration period in seconds.
rollover specifies the key rollover period in seconds.
sigval specifies the signature validity period in seconds.
sigPub specifies the signature publication period in seconds.
Example
Request 
 
https://172.17.151.255/config/dnskeyimport?id=key777&type=zone&state=1&passphrase=6666&keysz=1024&hash=1&ttl=86400&exp=2419200&rollover=604800&sigval=604800&sigPUB=302400
