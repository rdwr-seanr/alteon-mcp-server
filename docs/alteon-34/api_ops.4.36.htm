Create Application Engine Log
REST Method 
POST
Description 
Creates the application engine log file.
Note: Allowing Application Services Trace Logging may impact performance on Alteon traffic processing capabilities. Make sure that you disable trace logging when you are done.
Alteon Relative URI Path 
Enable or disable logging:
/config/applog?/config/applog?tcp=<1/0>&compress=<1/0>&caching=<1/0>&ssl=<1/0>&http=<1/0>&httpmod=<1/0>&appshape=<1/0>&contclss=<1/0>&dtable=<1/0>
where:
— tcp enables and disables TCP logging.
— compress enables and disables compress logging.
— caching enables and disables caching logging.
— ssl enables and disables SSL logging.
— http enables and disables HTTP logging.
— httpmod enables and disables HTTPmod logging.
— appshape enables and disables AppShape logging.
— contclss enables and disables class logging.
— dtable enables and disables data table logging.
Clear log files:
/config/applog?clearlog
Example
Request 
 
https://172.17.151.225/config/applog?tcp=yes&compress=yes&caching=no&ssl=no&
http=yes&httpmod=no&appshape=no&contclss=no&dtable=no
Response 
 
{
"status":"ok",
"message":"200 OK - The request has succeeded"
