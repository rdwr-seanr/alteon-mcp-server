Create New Table Item
HTTP Method 
POST
Description 
Sends an HTTP POST request with a body containing JSON text describing the changed parameters of the new row. For a sample response, see JSON Data Formats.
Note: When adding an object with augmented tables (for example, a real server or a virtual service), send elements in the first table using an HTTP POST command (create), and send augmented table elements using an HTTP PUT command (edit). For more information about augmented tables, see Working with Augmented Tables.
Alteon Relative URI Path 
/config/<table name>/<table key>
Example 
Adds real server “real2” with client NAT settings of type “address/subnet” and real server weight 10.
Request (from the real server first part table) 
 
https://172.17.151.225/config/SlbNewCfgEnhRealServerTable/real2
Body 
 
{
"Index": "real2",
"ProxyIpMode":"2",
"ProxyIpAddr":"30.30.30.30",
"ProxyIpMask":"255.255.0.0"
}
Response 
 
{
"status":"ok"
}
