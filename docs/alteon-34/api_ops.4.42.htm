Using Indexes to Set Parameter Values
For parameters with multiple values, use the relevant index to set the required parameter value. For example, the group metric has the following possible values:
1=round robin
2=least connections
3=minmisses
4=hash
5=response
6=bandwidth
7=phash
8=svcleast
The index 4 sets the group metric “hash”.
