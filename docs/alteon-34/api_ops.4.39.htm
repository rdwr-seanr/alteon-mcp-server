Export Technical Support Log File
REST Method 
GET
Description 
Downloads the Technical Support dump file from Alteon to the client computer. The file is available as a regular attachment.
Alteon Relative URI Path 
/config/gettechdata
Example  
 
https://172.17.151.225/config/gettechdata
