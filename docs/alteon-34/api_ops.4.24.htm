Export Configuration
REST Method 
GET
Description 
Downloads a configuration file from Alteon to the client computer.
Alteon Relative URI Path 
/config/getcfg?pkey=<yes/no>&passphrase=<string>&mansync=<yes/no> where:
pkey enables and disables the use of private keys.
passphrase specifies the password to decrypt the private key.
mansync enables and disables exporting internal indexes.
Example  
 
https://172.17.151.225/config/getcfg?pkey=yes&passphrase=radware&mansync=yes
Response 
The reply is a downloaded tar.gz file. The following is example content for this file:
