Upload Software Image
REST Method 
POST
Description 
Imports software to Alteon from the client computer.
Alteon Relative URI Path 
/config/softwareimport?type=<adc/vadc/all>&adcimg = <1..maxId>& vadcimg=<1..maxId>&pass=<passwd> where:
type specifies whether the software that uploads contains ADC-VX-infrastructure software, ADC software, or both.
adcimg specifies the ADC image ID.
vadcimg specifies the vADC image ID.
pass specifies the password for the uploaded software version.
Example  
Uploads a vADC software image to slot 2.
Request 
 
https://172.17.151.225/config/softwareimport?type=adc&adcimg=2
Response 
 
{
"status":"ok",
"message":"200 OK - The request has succeeded"
"application message":"1/30/14 14:8 Image downloaded via Browser - Successful"
}
Alternative Method 
Use the following API calls to upload a software image to Alteon using FTP, TFTP, or SCP.
In each case, a "status":"ok" message signifies only that the upload process has begun. To verify that the upload process has finished, poll the MIB string agTftpLastActionStatus until the status is “successful”.
For VA/4408:
 
? PUT https://10.205.124.120:443/config
? Content-Type: application/json
{
"agCfgTftpHostIPVersion" : <IP version>,
"agTftpImageFileName" : <file path and name>,
"agTftpMode" : <transfer mode 1=TFTP, 2=FTP, 3=SCP>,
"agTftpPort" : <1=Data, 2=Management>,
"agTftpServer" : <server IP address>,
"agTftpUserName" : <FTP/SCP user name for when agTftpMode is FTP or SCP >,
"agTftpPassword" : <FTP/SCP password>,
"agTftpVersionPass" : <image upgrade password>,
"agTftpImage" : <image slot 2=Image1,3=Image2>,
"agTftpAction" : <action 2=Image get>
}
For VX/vADC/Standalone:
 
? PUT https://10.205.124.120:443/config
? Content-Type: application/json
{
"agCfgTftpHostIPVersion" : <IP version>,
"agTftpImageFileName" : <file path and name>,
"agTftpMode" : <transfer mode 1=TFTP, 2=FTP, 3=SCP>,
"agTftpPort" : <1=Data, 2=Management>,
"agTftpServer" : <server IP address>,
"agTftpUserName" : <FTP/SCP user name for when agTftpMode is FTP or SCP >,
"agTftpPassword" : <FTP/SCP password>,
"agTftpVersionPass" : <image upgrade password>,
"agTftpImageAdc" : <ADC image slot 1=Image1, 2=Image2, 3=Image3, 4=Image4, 5=Image5 … 10=Image10>,
"agTftpImageVx" : <VX image slot 1=Image1, 2=Image2, 3=Image3, 4=Image4>,
"agTftpImageType" : <image type 1=VX, 2=vADC, 3=All (VX + vADC)>
"agTftpAction" : <action 2=Image get>
}
