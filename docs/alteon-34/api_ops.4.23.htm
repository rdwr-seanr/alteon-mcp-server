Import Configuration Tar File
REST Method 
POST
Description 
Imports a configuration tar file to Alteon from the client computer.
Alteon Relative URI Path 
/config/configimport?pkey=<yes/no>&passphrase=<string> where:
pkey enables and disables the use of private keys.
passphrase specifies the password to decrypt the private key.
Note: The name of the file appears in the Request Payload section of the header.
Example
Request 
 
https://172.17.151.225/config/configimport?pkey=yes&passphrase=radware
Response 
 
{
"status":"ok",
"message":"200 OK - The request has succeeded",
"application message":"3/18/14 17:36 Get config : - Successful"
}
