Revert Alteon Configuration
HTTP Method 
POST
Description 
revert — Discards any pending configuration changes.
revertapply — Discards any configuration changes that have not yet been saved to memory.
Alteon Relative URI Path 
/config?action=revert or /config?action=revertapply
Examples
Request 
 
https://172.17.151.225/config?action=revert
 
https://172.17.151.225/config?action=revertapply
Response 
 
{
"status":"ok"
}
