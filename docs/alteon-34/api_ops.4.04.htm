Update Scalar
HTTP Method 
PUT
Description 
Updates Alteon configuration parameter values. Sends an HTTP PUT request with body containing JSON text describing the changed scalars. For a sample response, see JSON Data Formats.
Alteon Relative URI Path 
/config
Example 
Sets the value of the Default Domain Name parameter.
Request 
 
https://172.17.151.225/config
Body 
 
{
"dnsNewCfgDomainName":"radware.com"
}
Response 
 
{
"status":"ok"
}
