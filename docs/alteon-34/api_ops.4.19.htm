Get Alteon Configuration File
HTTP Method 
GET
Description 
Gets the Alteon configuration file (archive of all internal module configurations for AppWall and Alteon).
Alteon Relative URI Path 
/config/getcfg exports the configuration without private keys.
/config/getcfg?pkey=<yes/no>&passphrase=<string>&mansync=<yes/no> exports the configuration where the following optional parameters are specified:
pkey enables and disables the use of private keys. Allowed only for users with certificate administration privileges.
passphrase specifies the password to encrypt the private key.
mansync enables and disables exporting internal indexes.
Example  
 
https://172.17.151.225/config/getcfg?pkey=yes&passphrase=radware
