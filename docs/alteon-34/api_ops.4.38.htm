Create Technical Support Log File
REST Method 
POST
Description 
Creates the Technical Support dump file from Alteon to the client computer.
Alteon Relative URI Path 
/config/techdump?usekey=<yes/no>&passphrase=<string> where:
usekey enables and disables the use of private keys.
passphrase specifies the password for the private key.
Example
Request 
 
https://172.17.151.225/config/techdump?usekey=yes&passphrase=radware
Response 
Alteon responds with 200 OK to indicate that the technical support log file creation has started:
 
{
"status":"ok",
"message":"200 OK - The request has succeeded"
}
