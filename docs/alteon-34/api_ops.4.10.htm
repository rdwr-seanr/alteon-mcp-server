Delete Table Item
HTTP Method 
DELETE
Description 
Sends an HTTP DELETE request with an empty body.
Notes  
To delete an object that has augmented tables, it is sufficient to send the HTTP DELETE command with the name of the first table.
Radware recommends that you use the DELETE HTTP command to delete an object instead of the “DeleteStatus": integer // {1=OTHER, 2=DELETE} parameter.
Alteon Relative URI Path 
/config/<table name>/<table key>
Example  
Deletes real server “real2”.
Request 
 
https://172.17.151.225/config/SlbNewCfgEnhRealServerTable/real2
Response 
 
{
"status":"ok"
}
