Show Configuration Differences
HTTP Method 
GET
Description 
getdiff — Displays the difference between the running configuration and the pending configuration.
getdiffflash — Displays the difference between the running configuration and the saved configuration.
Alteon Relative URI Path 
/config/getdiff or /config/getdiffflash
Examples
Requests 
 
https://172.17.151.225/config/getdiff
 
https://172.17.151.225/config/getdiffflash
