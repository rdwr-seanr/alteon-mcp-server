Export GSLB DNSSEC Key
REST Method 
GET
Description 
Exports a DNSSEC key as text, or downloads the DNSSEC key as a file from Alteon to the client computer.
Alteon Relative URI Path 
/config/getgdnskey?id=<value>&src=<txt/file>&passphrase=<string>
passphrase specifies the password to decrypt the private key.
Example
Request 
 
https://172.17.151.225/config/getgdnskey?id=key44&src=txt&passphrase=6666
