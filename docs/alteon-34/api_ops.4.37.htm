Export Application Engine Log
REST Method 
GET
Description 
Downloads the application engine log file from Alteon to the client computer. The file is available as a regular attachment.
Alteon Relative URI Path 
/config/getapplog
Example  
 
https://172.17.151.225/config/getapplog
