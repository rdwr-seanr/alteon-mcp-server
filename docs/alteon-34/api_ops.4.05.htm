Get Successive Table Items
HTTP Method 
GET
Description 
Requests the contents of a table.
Returns JSON text describing the content of the table rows. For a sample response, see JSON Data Formats.
Alteon Relative URI Path 
/config/<table name> returns the entire table.
/config/<table name>?offset=<value>&count=<value>&props=<table element1>,<table element2>,... where the following optional parameters are specified:
— offset <index> specifies the next index to read from. If no value is specified, Alteon reads from the beginning of the table.
— count specifies the number of rows to read.
— props specifies the columns to read from each entry. If no value is specified, Alteon reads all columns.
Example 
Retrieves the first 20 rows of the VLAN table and specifies a list of required columns per VLAN.
Request 
 
https://172.17.151.225/config/VlanNewCfgTable?count=20&props=State,VlanId,VlanName,Ports
Response 
 
{
"VlanNewCfgTable":[{
"State":"3",
"VlanId":"2",
"VlanName":"Default VLAN 2",
"Ports":"00:00:00:00:00:00:00:00",
},
{
"State":"3",
"VlanId":"3",
"VlanName":"VLAN 3",
"Ports":"60:00:00:00:00:00:00:00",
}]
}
