Export SSL Certificate
REST Method 
GET
Description 
Exports an SSL certificate as text, or downloads the SSL certificate as a file from Alteon to the client computer.
Alteon Relative URI Path 
/config/getcert?id=<value>&src=<txt/file>&type=<pair/srvcrt/key/inca/clca>&passphrase=<string>
id specifies the certificate ID
key specifies the key
pair specifies the cert+key
srvcrt specifies the server certificate
inca specifies the intermca
clca specifies the trusted certificate
passphrase specifies the password to decrypt the private key (required when type is set to key or pair).
Example
Request 
 
https://172.17.151.225/config/getcert?id=mycert&src=txt&type=key&passphrase=6666
